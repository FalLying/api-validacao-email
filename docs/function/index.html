    public sendEmail(req: Request, res: Response): Response source

    Essa função instanciará nossa class EmailVerification passando os dados necessários que são recebidos através da requisição http. A função funciona de forma sincrona, pois, o serviço de e-mail pode demorar algum tempo até ser concluído. Por fim, caso algum erro seja encontrado será retornado no console e a requisição será encerrada com um status 200.
